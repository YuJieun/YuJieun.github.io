1. 내풀이

```java
import java.util.*;

class Solution {
    public int[] solution(int[] progresses, int[] speeds) {
        ArrayList<Integer> arr = new ArrayList<>();
        Queue<Work> q = new LinkedList<Work>();
        for(int i=0; i<progresses.length; i++){
            q.offer(new Work(progresses[i],speeds[i]));
        }
        
        while(true){
            boolean flag = true;
            int cnt = 0;
            int len = q.size();
            for(int i=0; i<len; i++){
                Work tmp = q.poll();
                int new_progress = tmp.progress + tmp.speed;
                if(new_progress>=100 && flag == true){
                    cnt++;
                }else{
                    flag = false;
                    q.offer(new Work(new_progress,tmp.speed));
                }
            }
            if(cnt >0)
                arr.add(cnt);
            if(q.size()==0)
                break;
        }
        
        int[] answer = new int[arr.size()];
        for(int i=0;i<answer.length;i++){
            answer[i] = arr.get(i);
            System.out.print(answer[i]);
        }
        
        return answer;
    }
}

class Work{
    int progress;
    int speed;
    
    Work(int p, int s){
        this.progress = p;
        this.speed = s;
    }
}
```

class만들어서 나름 풀음.



2. 모범답안

```java
import java.util.ArrayList;
import java.util.Arrays;
class Solution {
    public int[] solution(int[] progresses, int[] speeds) {
        int[] dayOfend = new int[100];
        int day = -1;
        for(int i=0; i<progresses.length; i++) {
            while(progresses[i] + (day*speeds[i]) < 100) {
                day++;
            }
            dayOfend[day]++;
        }
        return Arrays.stream(dayOfend).filter(i -> i!=0).toArray();
    }
}
```

Arrays.stream을 이용해서 풀었다..어떻게 이렇게 짧게 풀지 ㅠㅠ



3. 모범답안2

   ```java
   import java.util.*;
   
   class Solution {
       public int[] solution(int[] progresses, int[] speeds) {
           Queue<Integer> q = new LinkedList<>();
           List<Integer> answerList = new ArrayList<>();
   
           for (int i = 0; i < speeds.length; i++) {
               double remain = (100 - progresses[i]) / (double) speeds[i];
               int date = (int) Math.ceil(remain);
   
               if (!q.isEmpty() && q.peek() < date) {
                   answerList.add(q.size());
                   q.clear();
               }
   
               q.offer(date);
           }
   
           answerList.add(q.size());
   
           int[] answer = new int[answerList.size()];
   
           for (int i = 0; i < answer.length; i++) {
               answer[i] = answerList.get(i);
           }
   
           return answer;
       }
   }
   ```
